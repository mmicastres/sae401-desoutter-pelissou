Routes :

Règle générale :

Mettre des status 1 c'est bon 0 c'est pas bon.

Get : 

	Albums :	


	get albums => get les n derniers albums par date ajout

	get albums/{idalbum} => get l'album n

	get categorie/{categorie}/albums => get les albums par categorie

	get artiste/{idartiste}/ablums => get les albums de l'artiste n

	get albums?validate=0 => get les albums pas encore validés


	Utilisateurs :


	get utilisateurs/{idutilisateur} => get les infos de l'utilisateurs n
	
	get utilisateurs?banni=1 => get tous les utilisateurs ban



	Commentaires :	


	get utilisateur/{idutilisateur}/commentaires => get tous les commentaires / notes de l'utilisateur n

	get album/{idalbum}/commentaires => get les commentaires sous l'album n

	get commentaires?valide=0 => get tous les commentaires non valides


	
	Titres :


	get albums/{idalbum}/titres => get les titres d'un album n

	get artiste/{idartiste}/titres => get les titres de l'artiste n

	
	
	Anecdotes :


	get titre/{idtitre}/anecdotes => get l'anecdote sur le titre n

	get anecdote?valide=0 => get les anecdotes pas validées

	
	Categories :


	get categories => get toutes les catégories



Post :
	 post incriptions => Données formulaire (Request) : pseudo, mail, mdp (date inscription ?), pp


	 post albums/{idalbum}/commentaires => Données form commentaire (Request) : pseudo, commentaire, note (optionelle), (date ajout ?).


	 post artistes/{idartiste}/albums => Données form album (Request) : nom, pochette, date création, categorie, pseudo


	 post albums/{idalbum}/titres ou post artiste/{idartiste}/titres => Données form titres (Request) : nom, pochette, paroles, lien écoute


	 post titres/{idtitre}/anecdotes => Données form anecdote (Request) : commentaire, (date ajout)

	 
	 post categories => Données form categorie (Request) : nom_categorie



Put : 
	
	put artistes/{idartiste}/albums =>  Données form album (Request) : nom, pochette, categorie
	
	put albums/{idalbum}/admin => album_valide


	put albums/{idalbum}/titres ou put artiste/{idartiste}/titres => Données form titres (Request) : nom, pochette, paroles, lien écoute

	put titres/{idtitre}/admin => titre_valide


	put utilisateurs/{idartiste}/album/{idalbum}/titres/{idtitre}/anecdotes/{idanecdote} => Données form anecdote (Request) : commentaire

	put admins/anecdote?valide=0/{idanecdote} => anec_valide



	put utilisateurs/{id_utilisateur} => Données form utili : pp, mdp
